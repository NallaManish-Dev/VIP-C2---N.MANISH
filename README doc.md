Book A Doctor
=============

A modern MERN Stack healthcare appointment booking platform that allows patients to search for doctors, view profiles, and book appointments online.

Project Overview
================

Book A Doctor is a full-stack web application developed using the MERN stack.

The platform helps patients connect with doctors and schedule appointments digitally.

Key Features:

*   User Registration
    
*   User Login
    
*   JWT Authentication
    
*   Doctor Listing
    
*   Doctor Search
    
*   Doctor Profile Page
    
*   Appointment Booking
    
*   Appointment Management
    
*   Admin Dashboard
    
*   Responsive UI
    
*   MongoDB Database Integration

Installation
============

Clone Repository
----------------

git clone

cd book-a-doctor

Backend Setup
=============

Navigate to backend folder:

cd backend

Install dependencies:

npm install

Create .env file:

PORT=5000

MONGO\_URI=your\_mongodb\_connection\_string

JWT\_SECRET=your\_secret\_key

Start backend server:

npm run dev

Frontend Setup
==============

Navigate to frontend folder:

cd frontend

Install dependencies:

npm install

Start frontend:

npm run dev

=====================
